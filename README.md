# Semnastika 2026 - Landing Page 🚀

**Semnastika (Seminar Nasional Informatika)** adalah landing page resmi untuk seminar nasional yang diselenggarakan oleh HIMATIK Universitas Alma Ata. Proyek ini dibangun dengan fokus pada aspek **Front-End Development** untuk memberikan informasi detail terkait perkembangan *Computer Vision* di era AI secara interaktif dan responsif.

---

## 📄 Deskripsi Proyek

Website ini dirancang sebagai pusat informasi dan gerbang pendaftaran bagi para peserta seminar. Dengan mengusung tema **"Perkembangan Computer Vision di Era AI"**, desain antarmuka menekankan estetika teknologi modern, profesionalisme akademis, dan kemudahan navigasi.

Website ini akan dipublikasikan pada subdomain resmi: [senastika.almaata.ac.id](https://senastika.almaata.ac.id)

## ✨ Fitur Utama

*   **Responsive Design:** Optimal diakses melalui perangkat mobile, tablet, maupun desktop.
*   **Modern Hero Section:** Header yang menarik dengan Headline yang kuat dan Call-to-Action (CTA) yang jelas.
*   **Speaker Profiles:** Section khusus yang menampilkan profil pembicara ahli menggunakan layout kartu yang elegan.
*   **Event Information:** Informasi jadwal, waktu, dan platform (Zoom) yang mudah dipahami.
*   **Interactive Elements:** Animasi hover dan transisi halus menggunakan CSS murni untuk meningkatkan pengalaman pengguna.

## 🛠️ Tech Stack

Proyek ini dibangun menggunakan teknologi *standard-compliant* Front-End murni:

*   **HTML5:** Struktur semantik untuk SEO dan aksesibilitas yang lebih baik.
*   **CSS3:** Styling kustom dengan Flexbox dan Grid System untuk tata letak yang presisi.
*   **Google Fonts:** Menggunakan font modern (Montserrat/Poppins) untuk keterbacaan yang optimal.
